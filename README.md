  A aplicação foi desenvolvida na linguagem Python, utilizando API
REST por meio da FastAPI, Redis como memória rápida e RabbitMQ como
fila de tarefas. O Frontend da aplicação foi desenvolvido em HTML, utilizando
CSS e JavaScript.
  O objetivo da aplicação é ser um Guia de Viagem Inteligente, o usuário
irá digitar o nome da cidade que deseja conhecer, o servidor irá retornar a
temperatura que está na cidade neste determinado momento e irá gerar um
guia em PDF com dicas para a viagem.

Uso de Inteligência Artificial:
Durante o desenvolvimento do projeto, utilizei IA para:
* Auxílio na interpretação de erro do Docker 
* Auxílio para compreensão de termos técnicos
* Auxílio na compreensão geral do trabalho
